# LockeWeb Companion

Webapp companion local para RandomLocke/Nuzlocke de Pokémon Esmeralda con lectura en directo desde mGBA.

Pensada para usar en una segunda pantalla mientras juegas. La app muestra información útil de la run y permite registrar capturas rápidamente en una checklist de Hoenn.

## Funciones principales
- Lectura en directo desde mGBA mediante script Lua.
- Bridge local en Node.js.
- Equipo actual en directo:
  - mote;
  - especie real;
  - sprite;
  - nivel;
  - PS;
  - stats;
  - ataques;
  - PP.
- Ficha de entrenador:
  - nombre;
  - ID;
  - dinero;
  - medallas.
- Mochila leída desde mGBA.
- Cementerio automático usando la Caja 14 del PC.
- Registro rápido de capturas.
- Checklist de rutas y zonas de Hoenn.
- Buscador de Pokémon con tolerancia a errores de escritura.
- Catálogo local offline de Pokémon de Gen 1, Gen 2 y Gen 3.
- Guardado automático en navegador.
- Exportar/importar backup JSON.
- Interfaz responsive.
- Inicio fácil en Windows con archivos `.bat`.
